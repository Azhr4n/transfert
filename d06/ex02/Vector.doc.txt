
<- Vector ----------------------------------------------------------------------
The Vector class handles Vector.

Instance are created by specifying the destination ('dest') vertex and
optionnaly the origin ('orig') Vertex.

new Vector( array('dest'=>(Vertex)$vertex) );

The class provides the following methods :
	The class provides the following methods :
	float	getX(), getY()...,
	float	magnitude(), return the norme,
	Vector	normalize(), return a normalized vector,
	Vector	add(Vector $rhs), return the sum of two vector,
	Vector	sub(Vector $rhs), return the substraction of two vector,
	Vector	opposite(), return the opposite vector of the actual vector,
	Vector	scalarProduct($k), return the scalar product
	float	dotProduct(Vector $rhs), return the dot product of two vector,
	Vector	crossProduct(Vector $rhs), return the scalar product of two vector,
	float	cos(Vector $rhs), return the value of the angle between two vector.
